## Network Intrusion Detection – UNSW-NB15

This project applies Machine Learning techniques to detect network intrusions using the UNSW-NB15 dataset.
The workflow focuses on:

1. **Data preprocessing**
2. **Feature reduction using PCA**
3. **Testing ML algorithms**
4. **Evaluating and comparing results**
---


## Dataset

UNSW-NB15 – Australian Centre for Cyber Security (ACCS).
Contains benign traffic + 9 attack categories.
